                                function getValues() {
    
                                    let startValue = document.getElementById("startValue").value;
                                    let endValue = document.getElementById("endValue").value;
                                
                                    //validate data
                                    startValue = parseInt(startValue);
                                    endValue = parseInt(endValue);
                                
                                    if( Number.isInteger(startValue) && Number.isInteger(endValue) ) {
                                        
                                        let numbers = generateNumbers(startValue, endValue);
                                        displayNumbers(numbers);
                                    }
                                    else {
                                
                                        alert("Error: You must enter integers");
                                
                                    }
                                
                                }
                                
                                function generateNumbers(startValue, endValue) {
                                
                                    let numbers = [];
                                    
                                    for(let i = startValue; i <= endValue; i++) {
                                        
                                        if(i % 3 == 0 && i % 5 == 0) {
                                            numbers.push("fizzbuzz");
                                            continue;
                                        }
                                
                                        if(i % 3 == 0) {
                                            numbers.push("fizz");
                                            continue;
                                        }
                                
                                        if(i % 5 == 0) {
                                            numbers.push("buzz");
                                            continue;
                                        }
                                
                                        else {
                                            numbers.push(i);
                                        }
                                
                                    }
                                
                                    return numbers;
                                
                                }
                                
                                function displayNumbers(numbers) {
                                
                                    let templateRows = "";
                                
                                    for(var i = 0; i < numbers.length; i++) {
                                
                                        let number = numbers[i];
                                
                                        templateRows += `< tr >< td >${number}< td/>< t />`;
                                
                                    }
                                
                                    document.getElementById("results").innerHTML = templateRows;
                                
                                }

The Design:

To keep the code as clean as possible, the program is encapsulated into three functions:

  • getValues

    • This function serves as the entry point to the program, being the function called by the button's click event listener. First, it stores the user input into variables by accessing the value property from the startValue and endValue elements. Next, it castes these variables from string to number types, and validates that they are integers. Next, it creates a local array called numbers and assigns it the array returned by the generateNumbers function, passing it the startValue and endValue variables. Lastly, it calls the displayNumbers function, passing it the newley generated numbers array as it's argument.

  • generateNumbers

    • This function serves as the main logic of the program, being responsible for generating the numbers array and replacing the correct numbers with "fizz", "buzz", or "fizzbuzz" to be displayed on the screen. It takes two arguments(startValue, endValue) and uses a for loop to loop from that startValue to the endValue. In every iteration of the loop, it uses a conditional statement to check if the index value(i) is a multiple of 3, 5 or both 3 & 5. If it is a multiple of one of those values, it will push the corresponding "fizz", "buzz", or "fizzbuzz" to the numbers array and continue to the next iteration of the loop. The expected output should be a combination of numbers between the startValue and endValue mixed in with "fizz", "buzz", and "fizzbuzz" based on each number's multiples. Finally, it returns this numbers array.

  • displayNumbers

    • This function is responsible for modifying the html table on the page and inserting a new row with the corresponding number, "fizz", "buzz", or "fizzbuzz". The only argument in this function is numbers which is the finalized array of numbers passed in from the generateNumbers function. First, the templateRows variable is declared as an empty string. Next, it loops from 0 to the length of the numbers array. Within the loop, the number variable is declared being assigned to the current number in the numbers array. It then adds a template literal to the templateRows variable with the necessary HTML for a table row and table data element, containing the number variable. This process is then repeated until the loop is finished. Lastly, it uses the getElementById method to assign the element's innerHTML property with an id of "results" to the value of templateRows.
